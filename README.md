Project Files - Marraja Brotherhood
This repository contains the HTML files for the website of the Agrupación de la Santa Agonía, Vera Cruz y Condena de Jesús (Marraja Brotherhood). Below is a detailed description of each file's function.

File Structure
File	Description
Index.html	Main page of the website. Entry point of the site.
AGONIA.html	Page for the Santa Agonía section (one of the brotherhood's divisions).
CONDENA DE JESUS.html	Page for the Condena de Jesús section.
VERA CRUZ.html	Page for the Vera Cruz section.
Contact.html	Contact page with email, phone, address, etc.
Directors.html	Page displaying the board of directors and responsible members.
Documents.html	Archive of historical and official documents (magazines, regulations, etc.).
Multimedia.html	Multimedia archive (photos and videos organized in Google Drive folders).
Calendar.html	Calendar of events and rehearsals (integrated with Google Calendar).
History.html	History of the brotherhood or historical section.
Header.html	Reusable header component.
Footer.html	Reusable footer component.
QR - copy.html	Page with QR code or related information (backup copy).
Note: Header.html and Footer.html are partial components included in other pages.

Technologies Used
HTML5 - Semantic structure

CSS3 / Tailwind CSS - Styles and responsive design

JavaScript - Interactivity and dynamic functionality

FullCalendar - Event calendar visualization

Google Calendar API - Real-time event loading

Google Fonts - Typography (Inter, Playfair Display, Material Symbols)

How to Use
Clone the repository:

bash
git clone https://github.com/your-username/your-repository.git
Open Index.html in your browser to view the main page.

For development, you can use a local server such as Live Server in VS Code.

Developer Notes
Header.html and Footer.html are components included via JavaScript or SSI (Server Side Includes) in the main pages.

Google Drive URLs in Documents.html and Multimedia.html must be updated with the correct folder and file IDs.

The Google Calendar API key in Calendar.html must be configured with a valid key.
